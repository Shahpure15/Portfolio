BloodAid - Blood Donation Platform

BloodAid is a web application designed to connect blood donors with recipients in real-time. It provides an easy-to-use platform where users can register, find blood donors based on their location, and contact them instantly. The platform ensures a smooth and secure process for both donors and seekers.

The application is built using the MERN stack (MongoDB, Express.js, React.js, Node.js) and incorporates Google and LinkedIn authentication for secure user access. It also features a dynamic donor search, secure messaging, an interactive dashboard, and an emergency request feature for urgent blood needs.

BloodAid aims to make blood donation easier and more accessible, ultimately saving lives and fostering a community of altruism and support.
